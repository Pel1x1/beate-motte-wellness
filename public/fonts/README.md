# Инструкция по добавлению шрифтов

## Куда добавить файлы шрифтов

Добавьте ваши файлы шрифтов в эту папку: `public/fonts/`

## Какие файлы нужны

### Основной шрифт (MainFont)
Это шрифт для всего основного текста на сайте. Назовите файлы:
- `main-font.woff2` (предпочтительный формат)
- `main-font.woff` (запасной формат)

### Контрастный/Акцентный шрифт (AccentFont)
Это шрифт для заголовков и названия "JungebadMoscow". Назовите файлы:
- `accent-font.woff2` (предпочтительный формат)
- `accent-font.woff` (запасной формат)

## Где использовать шрифты

После добавления файлов в эту папку, шрифты автоматически применятся:

- **font-main** - основной шрифт (применяется ко всему тексту по умолчанию)
- **font-accent** - контрастный шрифт (используется для названия "JungebadMoscow" в шапке и футере)

## Как конвертировать шрифты в WOFF/WOFF2

Если у вас есть шрифты в других форматах (TTF, OTF), используйте онлайн конвертеры:
- https://transfonter.org/
- https://convertio.co/ru/font-converter/

## Пример структуры папки после добавления шрифтов

```
public/fonts/
├── README.md (этот файл)
├── main-font.woff2
├── main-font.woff
├── accent-font.woff2
└── accent-font.woff
```

## Дополнительная настройка

Если вы хотите использовать контрастный шрифт в других местах, добавьте класс `font-accent` к нужному элементу:

```tsx
<h1 className="font-accent">Заголовок с акцентным шрифтом</h1>
```
